# 개인정보위 사고 유형 트렌드 분석 대시보드

개인정보보호위원회 보도자료를 기반으로 개인정보 침해 사고의 유형과 트렌드를 분석하는 Streamlit 대시보드입니다.

## 주요 기능

- 사고 유형별 연도 추이 (라인차트)
- 월별 발생 건수 시계열 분석
- 연도별 키워드 빈도 변화
- 클러스터별 핵심 키워드 분류
- CSV 업로드로 데이터 업데이트 및 이력 관리
- 보도자료 원문 링크 연결

## 실행 방법

pip install streamlit plotly pandas</br>
streamlit run dashboard.py

## 데이터

개인정보보호위원회 보도자료를 수집 후 TF-IDF + K-Means 클러스터링으로 사고 유형을 분류한 CSV 파일을 사용합니다.
